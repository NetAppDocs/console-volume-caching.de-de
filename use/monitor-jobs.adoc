= Überwachen von Volume-Caching-Jobs auf der Seite „Audit“
:hardbreaks:
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können alle Volume-Caching-Jobs überwachen und ihren Fortschritt mithilfe der Audit-Seite der NetApp Konsole ermitteln.

.Schritte
. Wählen Sie im NetApp -Konsolenmenü *Administration* > *Audit*.
. Filtern Sie optional nach Zeit, Dienst, Aktion, Agent, Ressource, Benutzer oder Status.
. Erkunden Sie alle Caches und Vorgänge.
